# D-CUBE Project 도메인 용어집 — 부록
> 기존 회의 요약 36건에서 자동 추출 (2026-07-17). 본파일(dcube-도메인용어.md) 외 잔여 용어.

## 공정명
- **위탁임가공** [공정명]: 당사가 풀하드 등 원자재를 임가공업체에 위탁해 가공 후 완제품을 회수하는 방식.
- **수탁임가공** [공정명]: 외부로부터 가공을 위탁받아 수행하는 임가공. ERP의 PR/PO 프로세스로 처리 가능.
- **하역신고** [공정명]: 선박에서 원자재를 하역했음을 신고하는 통관 절차.
- **단판** [공정명]: 사양 변경 시 발생하는 소량 시험 생산. 월 30시간 이상 소요되어 원가 왜곡 주원인.
- **올발주** [공정명]: 한·중·일 대상 해외 원료 발주 업무 전반을 지칭하는 사내 용어.
- **정정량발주** [공정명]: 아이템별 안전재고 소진 예상 시점에 발주하는 방식(전체 발주의 약 60%).
- **오더발주** [공정명]: 확정 주문의 재고 부족분을 자동 표시·처리하는 발주 방식.
- **앵커코팅** [공정명]: 필름 접착력 강화를 위해 앵커층을 도포하는 코팅 공정.
- **UV하드코팅** [공정명]: 자외선 경화 방식으로 필름 표면 경도를 높이는 코팅 공정.
- **합지** [공정명]: 두 겹 이상의 필름·소재를 접착해 하나로 합치는 공정.
- **프라이머** [공정명]: 후속 코팅과의 접착력 향상을 위한 표면 처리 공정.
- **숙성** [공정명]: 코팅 후 물성 안정화를 위해 일정 시간 보관하는 공정.
- **투패스** [공정명]: 동일 라인을 두 번 통과시켜 처리하는 라우팅 방식(현재 수동 중심).
- **소둔** [공정명]: 철강 열처리(어닐링) 공정. 배치ID·코일ID 혼용 사례가 언급됨.
- **종판** [공정명]: 출하 전 규격 충족 여부를 확인하는 최종 검사 공정.
- **계근(웨이트리스트)** [공정명]: 입고 물량 계량 절차. 반영 시 PO가 입고완료로 전환되는 것으로 추정.
- **로트 편성** [공정명]: 원료 정식 판매를 위해 판매 단위를 구성하는 작업.
- **제강** [공정명]: 철강을 제련하는 공정 부문. CM과 통합설비관리시스템을 공용 사용.
- **입동** [공정명]: 차량이 야드(적재동)에 진입하도록 요청·승인받는 절차.
- **프린트** [공정명]: 강판 표면에 무늬를 인쇄하는 공정이자 로스가 많은 제품 유형.
- **디지털 프린트** [공정명]: 디지털 방식 무늬 인쇄 공정. 수량이 적을수록 원가 상승.
- **컬러(도장) 공정** [공정명]: 도장강판 처리 공정. 조색제 사용 여부에 따라 보세·내국작업 판정.
- **코일링** [공정명]: 스크랩을 투입해 처리하는 공정 단계. 누락·주말 경과로 처리 지연 발생.

## 설비명
- **케미컬 코팅기** [설비명]: 외주 대응 시 분리 운영 가능한 화학 코팅 전용 설비.
- **프린트기** [설비명]: 외주 대응 시 분리 운영 가능한 인쇄 전용 설비.
- **워크롤** [설비명]: PLTCM에서 교환 작업이 자동 입력되는 압연 설비 부품.
- **용접기** [설비명]: PLTCM 라인의 용접 설비. 두께 조건이 스케줄 부합 여부에 영향.
- **SBC** [설비명]: 야드에서 운용되는 전동차 종류 중 하나.
- **TMA(티마)** [설비명]: 전동차 타입(규격)을 구분하는 명칭.
- **하이카** [설비명]: 미수거 코일 운반과 관련해 언급된 현장 장비.
- **크레인** [설비명]: 야드 자동화 핵심 물류 설비. 자동화 후 효율 향상·고장 감소 확인.
- **드로스** [설비명]: 도금 공정의 드로스(찌꺼기) 제거 자동화 로봇.
- **지게차 로봇** [설비명]: 물류 자동화용 무인 지게차 설비.
- **칼라 오븐** [설비명]: 도장강판 생산 시 작업자가 수기로 온도를 세팅하는 오븐.
- **계량대** [설비명]: 원료·제품 무게를 측정하는 물류 공정용 계량 시설.
- **무인화창고** [설비명]: 자동화되어 사람이 직접 실사할 수 없는 창고 설비.
- **사외창고** [설비명]: 외부 위탁 보관 창고. 로케이션 정보 미연계로 실사 시 코일 위치 파악 곤란.
- **스마트동** [설비명]: 무인화 적용으로 빈 번지 확인과 파레트·스키드 단위 재고 관리가 가능한 자동화 창고동.
- **PLC/HMI** [설비명]: 현장 제어반·화면 수준의 태그 표준화 대상 설비·인터페이스.
- **밀(Mill)** [설비명]: 원자재를 생산설비 단위로 구분 관리하는 기준 단위.
- **6·7호기** [설비명]: 레벨2 데이터가 다량 존재하나 활용되지 않는 라인 설비.

## 시스템명
- **BDC** [시스템명]: SAP가 보유한 비즈니스 데이터 클라우드(Business Data Cloud) 플랫폼.
- **D-BOX** [시스템명]: 사내 협업툴. 폴더 구조 혼란·속도 저하 등 사용성 문제 지속 제기.
(X)- **NES** [시스템명]: 표준판 촬영 사진을 분석해 컬러코드를 산출하는 사내 시스템.
- **E-Biz** [시스템명]: 임가공업체가 2·3·4급·스크랩 제품을 판매하는 전자상거래 사이트.
- **LP 엔진** [시스템명]: 지역·고객사별 수익성 연동으로 최적 이익 배분을 도출하는 선형계획 시뮬레이션 엔진.
(X)- **셀플랜** [시스템명]: 과거 원클릭 자동 소싱·매칭 기능을 제공하던 시스템(참고 사례).
- **키오스크** [시스템명]: 컬러 연결 메뉴로 원료 재고와 주문을 매칭해 보여주는 현장 시스템.
- **가상 간트** [시스템명]: 영업 내부 주문 우선순위 조율용 가상 스케줄링 화면(구상 단계).
- **SD** [시스템명]: 주문번호·행번호·품목 등 영업 필드를 다루는 SAP 영업(Sales & Distribution) 모듈.
- **BIS** [시스템명]: 기준정보 관리와 연결된 사내 시스템. SAP 업데이트 시 불편이 언급됨.
- **ASQ** [시스템명]: 품질 기준 체계 개편 범위에 포함되는 품질 관련 시스템.
- **OTS** [시스템명]: 주문번호·행번호 조회로 중복 주문 여부를 확인하는 기존 조회 시스템.
- **영업 포털** [시스템명]: 주문 등록부터 생산·출고 현황까지 한 화면 처리를 구상 중인 영업 신규 시스템.
- **EAM** [시스템명]: MDM에서 기준정보를 배포받아 사용하는 설비관리 시스템(Enterprise Asset Management).
- **EIS** [시스템명]: 경영진 참고용 경영정보 대시보드 체계. 실무자용 설계로 활용도 낮다는 평가.
- **디케어** [시스템명]: 안전 KPI 관리 지표/시스템. 활용도가 낮다고 평가됨.
- **통합설비관리시스템** [시스템명]: 제강·CM 공용 설비관리 시스템. MES 등 타 시스템 연동 부족 지적.
- **MEXIMO** [시스템명]: 제강 부문에서 쓰던 설비관리 플랫폼. 통합설비관리시스템에 대부분 반영.
- **SAP PM** [시스템명]: CM 부문에서 과거 사용하던 SAP 설비보전(Plant Maintenance) 모듈.
- **야드 종합상황판(현황도)** [시스템명]: 차량 입·출입과 작업 현황을 확인하는 야드 모니터링 화면. 2D 기반, 3D 업그레이드 검토.
- **FBS** [시스템명]: 야드 관리를 담당하며 본사 등록 정보 자동 연계가 필요한 시스템.
- **GMS** [시스템명]: 야드 관리 화면 UX 개선을 위해 도입 검토 중인 신규 UI 시스템.
- **덱사크루** [시스템명]: GMS와 함께 검토되는 신규 UI 도입 후보 솔루션.
- **IRIS** [시스템명]: MES·ERP와 함께 MDM·SSO 통합 인증 연계 대상인 사내 시스템.
- **DK 로그인** [시스템명]: 여러 시스템을 하나의 계정으로 통합 인증하기 위한 사내 SSO 구상.
- **스마트 야드** [시스템명]: 열·행 단위로 위치를 촘촘히 매핑해 관리하는 기존 야드 관리 체계.
- **PDM 스캔** [시스템명]: 스크랩 처리에서 코일 스캔으로 운반·이동야드 하치를 트리거하는 단계.
- **TDA** [시스템명]: 코일별 작업 이력을 조회하는 시스템. 숫자 클릭 시 세부 내역 표시.
- **APL** [시스템명]: 라인 편성 관리 시스템. 야드 자동화 잔여 항목으로도 언급됨.
- **APA** [시스템명]: 포장재 가공 실적이 반영되는 시스템(모듈).
- **부자재관리** [시스템명]: 구매분 부자재의 불출 일시를 조회하는 화면.
- **조건불검사실적조회** [시스템명]: 조건 불량(부적합) 검사 실적을 확인하는 조회 메뉴.
- **3D 야드맵** [시스템명]: 비스마트동 빈 번지 확인용 3D 야드 지도(현재 운영 중단).
- **전동차 이송관리** [시스템명]: 전동차 이송 관리 메뉴. 운반기기 메뉴로의 통합 요청 제기.
- **로케이터** [시스템명]: 스키드 등 자재의 위치·변경 이력을 추적·조회하는 관리 체계.
- **SAP T-code** [시스템명]: 라미나 필름 발주 정보를 수기 입력하는 SAP 거래화면.
- **마스터코드** [시스템명]: 표준항목 기반으로 영역값·담당자·사용처 등을 등록·관리하는 마스터 데이터 등록 화면. 밸리데이션 부재 한계.
- **품질사양** [시스템명]: 품질키 개편 후 설계원 실관리 항목만 모아 신설 예정인 항목 그룹(가칭).
- **데이터 플랫폼** [시스템명]: 분산 데이터를 수집·정제해 AI/LLM 활용 기반을 만드는 신규 구축 3대 과제 중 하나.
- **디큐브 마스터플랜** [시스템명]: '프로젝트' 용어 혼동을 피해 채택한 사내 범용 프로젝트관리(PM) 솔루션 명칭.
- **MGM** [시스템명]: 마스터데이터거버넌스. 도구·전담 조직·역할 세트가 갖춰져야 작동한다고 논의됨.
- **인터지스** [시스템명]: ERP 배선 정보 변경 시 자동 동기화 대상인 물류 연동 시스템.
- **BI** [시스템명]: D+1 이후 실적을 피벗·자유 분석하는 분석계 시스템. 운영계 MES와 역할 분리.
- **사이즈 어서트 대기** [시스템명]: 발주 관리의 사이즈 확정 대기 기능. 실무에서 거의 사용되지 않음.

- **핸드링마크** [일반]: 상·하차 시 코일에 발생하는 취급 흔적. 발생 야드 특정을 위해 촬영 확인.
- **성품차** [일반]: 완성품 운반 차량. 필요 시 반제품차로 전환 운용.
- **반제품차** [일반]: 반제품 운반 차량. 차량 전환 관리 대상.
- **원단위** [일반]: 제품·공정 단위당 자재·에너지 소요량을 나타내는 생산관리 지표.
- **성력화** [일반]: 인력 절감 중심 자동화·합리화 방향(회사가 지양하는 접근).
- **등대공장** [일반]: WEF 선정 첨단 제조 혁신 공장(라이트하우스 팩토리).
- **클래스 코드** [일반]: 자재 분류 코드 체계. 6자리에서 40자리 확장, 원자재/제품 분리 설계 결정.
