# 또박또박 기능 인벤토리

> 조사일 2026-06-29 · 소개 페이지·사용자 매뉴얼·스크린샷 작성을 위한 사전 기능 조사. 6개 소스(문서/화면/컴포넌트/API/AI잡/기획)에서 raw 338개 추출 후 합성.

**제품 정의:** 또박또박은 회의 음성을 실시간으로 받아쓰고(STT), 화자를 자동으로 구분하며, AI가 구조화된 회의록을 자동 작성·검색·질의응답까지 해 주는 **자체호스팅(셀프호스트) 팀 회의 어시스턴트**다.

**핵심 가치**
- **전 코퍼스 로컬·프라이버시 우선**: STT는 완전 로컬에서 실행되고 음성·전사·요약을 외부 클라우드에 보내지 않는다. SQLite WAL 로컬 DB로 별도 DB 서버 없이 운영 가능하다.
- **팀 자산화**: altalt 같은 개인용 도구와 달리 프로젝트·폴더·팀(약 20명) 단위로 회의록을 공유·비공개 전환하며 축적한다.
- **근거 기반 AI**: 요약·AI챗 문장마다 발화 시각·화자 인용 마커를 달고 클릭하면 원본 오디오로 정확히 점프해 환각 신뢰 문제를 직격한다.
- **교차 회의 AI Q&A**: 한 회의를 넘어 폴더·프로젝트 전체를 가로질러 의미검색(임베딩+FTS) 기반으로 묻고 답한다.
- **한국어·도메인 특화**: 9개 언어 인식, 폴더별 오타사전, 명함 OCR 참석자 등록 등 한국어 사내 회의 실수요에 맞춘 고유 기능을 갖췄다.

## 소개 페이지용 핵심 셀링포인트 Top 8

1. **전 코퍼스 로컬·자체호스팅** — 음성·전사·요약 전체가 팀 서버/내 기기에만 머문다. 어떤 클라우드 경쟁사도 못 내는 차별점으로, 민감한 사내 회의를 안심하고 맡길 수 있다.
2. **근거를 클릭하면 그 순간으로 점프하는 AI** — 요약·챗 문장마다 발화 시각·화자 인용을 달아 원문 오디오로 정확히 이동. 한국어 STT의 불완전함과 AI 환각 신뢰 문제를 정면 해결(Granola/Fireflies급 희소 기능).
3. **폴더·프로젝트 교차 AI Q&A** — 한 회의를 넘어 여러 회의를 가로질러 묻고 답한다(의미검색+FTS). 경쟁사가 2026에 막 따라오는 영역을 선점.
4. **팀 자산화 + 실시간 공유** — 개인 도구와 달리 팀(~20명) 단위 공유·비공개 전환, 라이브 뷰어·호스트 위임으로 회의록을 함께 본다.
5. **회의 유형별 맞춤 회의록 + Mermaid 시각화** — 스탠드업·브레인스토밍·인터뷰 등 10여 종 양식과 자동 다이어그램으로 그냥 받아쓰기가 아닌 '읽기 좋은 회의록'.
6. **한국어·도메인 특화** — 9개 언어 인식, 폴더별 오타사전으로 고유명사·전문용어 일괄 교정. 명함 OCR로 참석자 자동 등록(경쟁사 카탈로그에 없는 고유 기능).
7. **GPU 없는 PC부터 데스크톱·모바일까지** — Apple Silicon/NVIDIA/CPU 환경별 STT 엔진 자동 선택, 시스템 오디오 캡처, 모바일 잠금화면·완전 오프라인 온디바이스 녹음.
8. **모델 선택 자유 + 인터넷 없이도** — 사용자별 LLM(Anthropic/OpenAI/Ollama/로컬 CLI)을 직접 고르고, 로컬 LLM으로 인터넷 없이 요약. 자체호스팅이라 모델 자유가 오히려 강점.

## 일반 사용자 매뉴얼 목차 제안

1. **시작하기**
   - 또박또박이란 / 데스크톱 앱 설치(macOS·Windows·Linux)
   - 첫 실행: 로컬 모드 vs 서버 모드 선택, 초기 셋업(환경 확인·자동설치)
   - 회원가입·로그인, 프로젝트 선택 화면 이해
2. **프로젝트와 폴더 정리**
   - 개인 프로젝트('내 회의')와 팀 프로젝트
   - 폴더 만들기·이동·중요 표시, 태그로 분류
   - 사이드바·프로젝트 스위처 사용법
3. **회의 녹음하기**
   - 새 회의 생성(제목·유형·참석자·이전 회의 참고)
   - 실시간 녹음: 시작·일시정지·재개·종료
   - 시스템 오디오 캡처(데스크톱), 모바일 백그라운드 녹음
   - 녹음 중 메모·북마크(Ctrl+B)·안건 첨부
4. **실시간 vs 파일 업로드(배치)**
   - 라이브 실시간 전사·AI 회의록 보기
   - 기존 오디오 파일 업로드 전사, 진행률 확인
   - 오프라인 온디바이스 녹음(모바일)
5. **AI 회의록 확인·편집**
   - 회의록 구조(핵심·논의·결정·Action Item) 읽기
   - 압축율 5단계·재구조화/증분 모드 조정
   - 블록 에디터로 직접 편집, 자연어 피드백으로 수정
   - 인용 마커·Mermaid 다이어그램·확장 팝업 활용
   - 회의록 재생성, 회의 잠금
6. **화자 정리**
   - 화자분리 켜기(설정·HF 토큰)
   - 화자 이름 바꾸기(화자 패널·전사 본문 더블클릭)
   - 화자 배지 클릭으로 발화 따라가기, 화자분리 재실행
7. **검색·AI 챗으로 활용**
   - 전역 검색·회의 내 검색
   - 회의에 질문(AI 챗), 스코프 전환(회의/폴더/프로젝트)
   - 폴더/프로젝트에게 묻기(교차 회의 Q&A)
   - 오타사전 만들기, 명함으로 참석자 등록
8. **공유·내보내기**
   - 공유 코드 발급·참여, 뷰어 모드, 호스트 위임
   - 프로젝트 멤버 초대(초대 링크)
   - 회의록 내보내기(MD/PDF/DOCX/프롬프트)
   - 회의·폴더·프로젝트 아카이브 export/import(이관·백업)
9. **예약·자동화**
   - 회의 예약(자동/수동 시작)·주간 반복
   - 데스크톱 백그라운드·트레이·자동시작
   - 다가오는·놓친 예약 안내
10. **설정·관리**
    - 개인 LLM 설정·연결 테스트, 로컬 LLM 연결
    - 회의 언어, STT 엔진, 화자분리·VAD 조정
    - 회의 템플릿·회의록 양식 관리, 테마, 비밀번호 변경
    - (관리자) 사용자 관리, 전역 설정
11. **휴지통·복구**
    - 삭제·복구·영구삭제·휴지통 비우기
