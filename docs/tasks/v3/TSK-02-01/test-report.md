## 커버리지 요약

테스트가 검증하는 주요 기능:

- **렌더링**: 탭 버튼 및 콘텐츠 DOM 마운트
- **기본 탭 선택**: defaultTab prop 동작
- **탭 전환**: 클릭 시 활성 탭 변경, aria-selected 속성
- **DOM 유지 전략**: 비활성 탭을 언마운트하지 않고 visibility/position으로 숨김
- **레이아웃**: flex-1 균등 배분, sticky 탭 바, 콘텐츠 영역 overflow
- **스타일링**: 활성/비활성 탭 시각적 구분 (border, text color)
- **제어/비제어 패턴**: controlled (activeTab + onTabChange) / uncontrolled 모드 모두 지원
- **상태 보존**: 탭 전환 시 이전 탭 콘텐츠가 DOM에 유지됨
