#### 테스트 목록 (모두 PASS)

1. 검색 입력란이 렌더링된다
2. 초기 상태에서 안내 문구를 표시한다
3. 검색 버튼 클릭 시 API를 호출한다
4. 검색 결과를 표시한다
5. 결과 클릭 시 회의 페이지로 이동한다
6. 검색 결과가 없으면 안내 문구를 표시한다
7. 동일 회의 결과가 그룹으로 묶여 표시된다
8. 그룹 헤더에 매칭 건수를 표시한다
9. 접기/펼치기 토글이 동작한다
10. 회의 헤더 클릭 시 해당 회의 페이지로 이동한다
11. 서로 다른 회의 결과는 별도 그룹으로 표시된다
